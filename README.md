# Digital Agency Website — Nexus

A pixel-perfect replication of the **"Digital Agency Website - Freebie"** by [Fauzan Ardhiansyah](https://www.figma.com/community/file/1058842196634115002/digital-agency-website-freebie), built with pure HTML, CSS & JavaScript.

## Live Preview
Open `index.html` in your browser or host on any static server.

## Sections
- **Navbar** — Sticky with blur effect + mobile hamburger menu
- **Hero** — Full-screen with animated browser mockup, floating badges, stat counters & skills ticker
- **Clients** — Logo strip with hover reveal
- **Services** — 3×2 grid with hover animations
- **Portfolio** — Filterable project grid (All / Web / Brand / App)
- **About** — Two-column layout with stacked image mockup & award badge
- **Testimonials** — 3-column card layout with highlighted middle card
- **CTA Banner** — Full-width with radial glow effect
- **Contact** — Two-column with details + contact form
- **Footer** — Brand column + 3-column nav grid

## Tech Stack
- **HTML5** — Semantic structure
- **CSS3** — Custom properties, Grid, Flexbox, CSS animations
- **Vanilla JavaScript** — Intersection Observer, scroll events, DOM manipulation
- **Fonts** — [Syne](https://fonts.google.com/specimen/Syne) (headings) + [DM Sans](https://fonts.google.com/specimen/DM+Sans) (body) via Google Fonts
